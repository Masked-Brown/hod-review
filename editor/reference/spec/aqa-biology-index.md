# AQA A-level Biology — content index (3.1 to 3.8)

**Board:** AQA A-level Biology (7402) · **Reference layer for `hod-review`**

> **What this file is.** The 8-topic map of AQA A-level Biology, scaffolded so the editor knows the shape of the whole subject and every finding can name the topic it sits in. **One subtopic is built to full depth** — the validated topic, **3.2 Cell structure** (`3.2-cell-structure.md`) — because depth shown on a real run beats bulk shipped and never exercised (see the build plan, §M2.5). The remaining subtopics are listed with scope so the structure is legible and extensible; each is built to depth only when a lesson on it is reviewed.
>
> **Why scaffold-not-dump.** A block of notes no finding could ever cite is bulk, and bulk lowers the methodology score. This index carries exactly what a finding needs to place a lesson (topic code, title, one-line scope, depth status) and no more. When a new topic is reviewed, its `3.x-<slug>.md` file is built to the same depth as cell structure, from the same corpus.
>
> **How findings cite spec points.** A finding's `SPEC` field uses the handle convention `AQA <subtopic-code> — <handle>` (e.g. `AQA 3.2.1 — cell fractionation`). The convention and the full published handle list for the validated topic are defined in **`3.2-cell-structure.md`**. Each topic publishes its own handle list when built to depth.
>
> **Provenance.** Subtopic structure and titles from AB's `professor-clive` `05_notes` corpus (the content index) and AQA's published specification. Examiner-insight and exam-question layers are added per topic as it is built to depth.

## 3.1 — Biological molecules ○

Monomers/polymers, the biochemistry a cell is built from and the tests that identify them. Underpins nearly every later topic.

| Code | Subtopic | One-line scope |
|---|---|---|
| 3.1.1 | Monomers and polymers | Condensation/hydrolysis; monomer–polymer relationships |
| 3.1.2 | Carbohydrates | Monosaccharides, glycosidic bonds, starch/glycogen/cellulose; reducing-sugar tests |
| 3.1.3 | Lipids | Triglycerides, phospholipids, saturated/unsaturated; emulsion test |
